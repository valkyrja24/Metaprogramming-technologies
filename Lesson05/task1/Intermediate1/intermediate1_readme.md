# ConfigParser

## Опис
`ConfigParser` — парсер рядків конфігурації та файлів у форматі `ключ=значення`. Підтримує пропуск порожніх рядків та рядків-коментарів (`#`). Помилки рядків обгортаються у власний `ConfigFormatException` із номером рядка та `InnerException`.

## Як запустити
1. Відкрити проєкт у Visual Studio або будь-якому IDE для C#.
2. Створити файл `settings.txt` з конфігурацією, наприклад:
```
username=admin
password=12345
# це коментар
port=8080
```
3. Запустити програму. Вона прочитає файл і виведе ключі та значення:
```
username = admin
password = 12345
port = 8080
```

## Особливості
- Пропускає порожні рядки та рядки, що починаються з `#`.
- Кожну помилку рядка обгортає у `ConfigFormatException` із номером рядка.
- Повертає `Dictionary<string,string>` з ключами та значеннями.

## Юніт-тести / тестові випадки
1. Коректний рядок: `"username=admin"` → повертає `("username", "admin")`.
2. Порожній рядок: `""` → кидає `ArgumentNullException`.
3. Рядок без `=`: `"username"` → кидає `FormatException`.
4. Рядок з пробілами: `"  user  =  123  "` → повертає `("user", "123")`.
5. Файл з коментарями та порожніми рядками обробляється коректно.
